Wave Framework incorporates the following open source components that are not required for the functionality of the framework, but are included to assist the developer for bootstrapping reasons.

jQuery

  • /resources/jquery.js
  • /resources/jquery2.js
  • jQuery website
  • One of the most popular JavaScript frameworks, jQuery is mainly used for AJAX, but can be incorporated for JavaScript UI programming for other reasons too.
  • This component is released under either the MIT License or the GNU General Public License Version 2.

YUI Reset CSS

  • resources/reset.css
  • YUI Library website
  • Reset CSS is used also optional, but recommended for UI design. Reset CSS files remove custom styles of different browsers, making sure that the rules applied by the developer are consistent across all browsers.
  • This component is released by Yahoo! Inc. under BSD License.

Adminer

  • /tools/adminer.php
  • Adminer website
  • Adminer database administration tool is not incorporated in framework in any way, but is included for easy database administration for developers.
  • This component is released by Jakub Vrana under Apache License Version 2 and GNU General Public License Version 2.
